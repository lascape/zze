# 🚀 快速启动指南

> 5分钟内运行微信小程序

## 📋 前置条件

1. **已安装 Node.js**（用于安装依赖）
2. **已安装微信开发者工具**
   - 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 💡 快速检查环境

在开始之前，可以运行环境检查脚本：

**Linux/Mac：**
```bash
bash check-env.sh
```

**Windows：**
```cmd
check-env.bat
```

这个脚本会自动检查：
- ✅ Node.js 和 npm 是否已安装
- ✅ 项目依赖是否已安装
- ✅ 项目文件是否完整

## 🎯 三步启动

### 第一步：安装依赖

在项目根目录打开终端，运行：

```bash
npm install
```

> ⏱️ 预计耗时：30秒 - 1分钟

### 第二步：导入项目到微信开发者工具

1. **打开微信开发者工具**

2. **点击"导入项目"或"+"**

3. **填写项目信息：**
   - 📁 **项目目录**：选择本项目的根目录（zze 文件夹）
   - 🆔 **AppID**：选择"测试号"（如果没有正式 AppID）
   - 📝 **项目名称**：zze 社区信息板（可自定义）

4. **点击"导入"**

### 第三步：构建 npm

这一步非常重要，必须执行才能使用 TDesign 组件库！

1. 在微信开发者工具顶部菜单栏：
   ```
   工具 → 构建 npm
   ```

2. 等待构建完成，会看到提示：
   ```
   ✅ 构建 npm 成功
   ```

3. 点击工具栏的 **"编译"** 按钮

4. 🎉 **成功！** 现在可以在模拟器中看到小程序运行了

## 🎨 预期效果

运行成功后，您会看到：

- **底部 TabBar**：社区、发现、发布、消息、我的（5个标签）
- **社区页**：显示当前社区名称、分类宫格、信息流列表
- **可以点击**：各个分类、帖子详情、社区切换等

## ⚙️ 项目设置（可选但推荐）

点击右上角 **"详情"** 按钮，在"本地设置"中勾选：

- ✅ 不校验合法域名、web-view...（推荐）
- ✅ 启用 ES6 转 ES5
- ✅ 启用增强编译

## 🧪 快速测试

### 测试发布功能

1. 点击底部 **"发布"** Tab

1. 点击 **"社区"** Tab
2. 点击顶部的社区名称
3. 选择其他社区
4. 返回后列表会自动更新

## ❓ 常见问题

### Q1: 构建 npm 失败怎么办？

**解决方法：**
```bash
# 1. 删除 node_modules 文件夹
rm -rf node_modules

# 2. 重新安装依赖
npm install

# 3. 在微信开发者工具中再次构建 npm
```

### Q2: 页面显示空白或报错？

**检查以下几点：**
1. 是否执行了 `npm install`
2. 是否执行了"构建 npm"
3. 是否点击了"编译"按钮
4. 查看控制台是否有错误信息

### Q3: TDesign 组件不显示？

**原因：** 未构建 npm

**解决：** 在微信开发者工具中执行 `工具 → 构建 npm`

### Q4: 找不到模块错误？

**解决方法：**
1. 确认 `miniprogram_npm` 目录是否存在
2. 重新构建 npm
3. 重新编译项目

### Q5: 提示 TDesign 组件路径错误？

**错误信息：**
```
app.json: ["usingComponents"]["t-toast"] 未找到组件
```

**原因：** 这是一个已知问题，已在最新版本修复。

**解决方法：**
1. 确保使用最新代码（已移除全局组件配置）
2. 重新构建 npm
3. 重新编译项目

**详细说明：** 查看 [TROUBLESHOOTING.md](./TROUBLESHOOTING.md)
