# 系统头像资源说明

## 系统头像

请在此目录下添加以下头像文件（建议尺寸：200px × 200px，PNG 格式）：

- `avatar-1.png` - 系统头像 1
- `avatar-2.png` - 系统头像 2
- `avatar-3.png` - 系统头像 3
- `avatar-4.png` - 系统头像 4
- `avatar-5.png` - 系统头像 5
- `avatar-6.png` - 系统头像 6
- `avatar-7.png` - 系统头像 7
- `avatar-8.png` - 系统头像 8
- `default.png` - 默认头像

## 头像风格建议

- 使用抽象/卡通风格的头像
- 避免使用真实人物照片
- 保持风格统一
- 背景透明或纯色

## 头像来源推荐

- [Avataaars Generator](https://getavataaars.com/)
- [UI Faces](https://www.uifaces.co/)
- [DiceBear Avatars](https://avatars.dicebear.com/)
- 自行设计几何图形头像

## 临时解决方案

在没有头像资源的情况下，系统会使用占位符路径，虽然图片无法显示，但不影响功能运行。
