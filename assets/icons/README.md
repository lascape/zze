# 图标和头像资源说明

## TabBar 图标

请在此目录下添加以下图标文件（建议尺寸：81px × 81px，PNG 格式）：

### 普通状态图标
- `community.png` - 社区图标
- `discovery.png` - 发现图标
- `publish.png` - 发布图标
- `message.png` - 消息图标
- `me.png` - 我的图标

### 选中状态图标
- `community-active.png` - 社区图标（选中）
- `discovery-active.png` - 发现图标（选中）
- `publish-active.png` - 发布图标（选中）
- `message-active.png` - 消息图标（选中）
- `me-active.png` - 我的图标（选中）

## 临时解决方案

如果暂时没有图标资源，可以：

1. 使用 iconfont 等网站生成图标
2. 使用微信官方图标库
3. 使用纯色方块作为占位符
4. 使用 TDesign 图标系统（需要调整 TabBar 配置）

## 图标来源推荐

- [iconfont](https://www.iconfont.cn/)
- [阿里巴巴矢量图标库](https://www.iconfont.cn/)
- [TDesign 图标](https://tdesign.tencent.com/miniprogram/components/icon)
- [Iconify](https://icon-sets.iconify.design/)
