═══════════════════════════════════════════════════════════════
                    如何运行这个微信小程序
═══════════════════════════════════════════════════════════════

您问："接下来怎么运行？"

答案很简单，只需 5 步：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第 1 步：安装依赖
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在项目目录打开终端，运行：

    npm install

⏱️ 需要 30 秒左右

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第 2 步：打开微信开发者工具
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 下载并安装微信开发者工具（如果还没有）：
   https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

2. 打开微信开发者工具

3. 点击"导入项目"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第 3 步：导入项目
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

填写项目信息：

    项目目录：选择本项目文件夹（zze）
    AppID：    点击"测试号"
    项目名称：  随便填，如"社区信息板"

然后点击"导入"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第 4 步：构建 npm（重要！）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在微信开发者工具顶部菜单：

    工具 → 构建 npm

等待提示："✅ 构建 npm 成功"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第 5 步：编译运行
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

点击工具栏的"编译"按钮

🎉 完成！在模拟器中查看小程序

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

您会看到什么？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ 底部有 5 个标签：社区、发现、发布、消息、我的
✓ 可以点击所有功能进行测试
✓ 可以发布信息（会自动审核）
✓ 可以私信、收藏、切换社区

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

快速测试
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

试试发布功能：
1. 点击底部"发布"
2. 选择"租房"
3. 填写表单提交
4. 查看"我的发布"（显示"审核中"）
5. 等 5 秒后变成"已通过"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

遇到问题？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

问：页面显示空白？
答：确认执行了"构建 npm"，然后点击"编译"

问：TDesign 组件不显示？
答：执行"工具 → 构建 npm"

问：构建失败？
答：删除 node_modules，重新 npm install
